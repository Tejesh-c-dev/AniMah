AniMah Project Features
=======================

1) Core Product Features
- Community platform for tracking anime, manhwa, and movies.
- Browse all titles with category tabs (All, Anime, Manhwa, Movie).
- Title discovery with search, sorting, and filtering.
- Personalized recommendations for authenticated users.
- Detailed title pages with rating summary and community reviews.

2) Authentication & User Account
- User registration and login.
- JWT-based authenticated API requests.
- Protected user-only functionality (reviews, favorites, watchlist, profile).
- One-time admin bootstrap endpoint (setup key based).
- Admin login flow.

3) Title Management
- Create new titles (anime/manhwa/movie) via authenticated form.
- Title metadata support: name, description, release year, type, genre.
- Cover image upload for titles.
- Public listing and per-title retrieval.
- Admin-only title deletion.

4) Reviews & Community Interaction
- Create, edit, and delete reviews.
- One-review-per-user-per-title UX guard in detail page.
- Review sorting (latest, top rated, most helpful).
- Review voting (helpful / not helpful).
- Reply to reviews.
- Review action rate limiting to reduce spam.

5) Watchlist Features
- Add title to watchlist with status.
- Update watchlist status.
- Remove title from watchlist.
- Watchlist status options include:
  PLAN_TO_WATCH, WATCHING, COMPLETED, DROPPED.
- Watchlist statistics endpoint.
- Watchlist view in user profile.

6) Favorites Features
- Add/remove favorites.
- Toggle favorite in one action.
- Check favorite status per title.
- Favorites tab in user profile.
- Interactive heart button with toast feedback.

7) User Profile Features
- Fetch and display current user profile.
- Profile image upload and update.
- Profile stats (reviews count, likes received).
- Profile tabs: watchlist, favorites, activity.
- Activity feed with mapped actions and timestamps.

8) Admin Dashboard Features
- Admin-only dashboard UI.
- View all users.
- Delete users (with role safeguards in UI logic).
- Promote/demote user roles (USER <-> ADMIN).
- View all titles and delete titles.

9) Frontend UX Features
- Responsive navigation with desktop and mobile menu.
- Light/Dark theme toggle.
- Route-based UI (home, login, register, add-title, profile, title detail, admin).
- Animated UI elements (Framer Motion transitions/effects).
- Loading skeletons and empty/error states across pages.
- Reusable UI components (StarRating, FavoriteButton, WatchlistDropdown, route guards).

10) API & Backend Features
- Express API modularized by domain: auth, titles, reviews, watchlist, favorites.
- Health endpoint: /api/health.
- Input sanitization middleware.
- Centralized error handling middleware.
- CORS configuration with localhost, configured origins, and Vercel deployment support.
- File upload middleware with type and size validation (JPEG/PNG/WebP/GIF, max 5MB).

11) Data & Infrastructure
- Prisma ORM integration with migrations.
- Supabase integration for storage/utilities.
- Build output separation for backend (dist-server).
- Scripts for dev, build, start, lint, and Prisma workflows.

12) Deployment & Environment
- Configurable environment variables for DB, auth, Supabase, admin setup, CORS.
- Next.js frontend + Express backend deployment flow documented.
- Vercel-oriented frontend deployment support with configurable API base URL.
